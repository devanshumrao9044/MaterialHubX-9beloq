## âŒ Common Mistakes

### "Invalid credentials" error

**Problem**: Trying to login without creating account first

**Solution**: 
1. Click **"Sign Up"** (not Sign In)
2. Create your account first
3. Then you can login

---

### "Admin password not working"

**Problem**: 
- Never created admin account, OR
- Using wrong password, OR
- Typo in email

**Solution**:
1. Make sure you created account with EXACTLY: `admin@materialhubx.com`
2. Use the password YOU set during sign up
3. Check for typos
4. Make sure email is exactly correct (no spaces, no capitals)

---

### "Can't see Admin Panel"

**Problem**: 
- Not using admin email, OR
- Typo in email during signup

**Solution**:
- Admin Panel only appears for: `admin@materialhubx.com`
- Create new account with exact email if you made typo
